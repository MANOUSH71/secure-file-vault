# 🔐 Secure File Vault Pro - Web Application

تطبيق ويب متقدم لتشفير وإدارة الملفات بأمان عالي مع دعم طرق تشفير متعددة ونظام مشاركة آمن.

## ✨ المميزات

### 🔒 طرق تشفير متعددة
- **AES-256-GCM**: معيار الصناعة، سريع وآمن (موصى به)
- **ChaCha20-Poly1305**: بديل حديث لـ AES، ممتاز للأجهزة المحمولة
- **Fernet**: بسيط وآمن، مبني على AES-128
- **RSA-2048**: تشفير غير متماثل، مثالي للملفات الصغيرة

### 📤 نظام المشاركة
- إنشاء روابط مشاركة آمنة
- تحديد مدة صلاحية الرابط
- حماية الروابط بكلمة مرور
- تتبع عدد مرات التحميل

### 💾 إدارة الملفات
- رفع وتشفير الملفات (حتى 100 ميجابايت)
- فك التشفير والتنزيل المباشر
- حذف الملفات
- عرض إحصائيات مفصلة

### 🎨 واجهة مستخدم عصرية
- تصميم Dark Mode جذاب
- واجهة عربية كاملة (RTL)
- Drag & Drop للملفات
- Responsive Design

### 🔐 الأمان
- تشفير قوي من الدرجة العسكرية
- PBKDF2 لاشتقاق المفاتيح (100,000 تكرار)
- ضغط البيانات قبل التشفير
- Zero-Knowledge Architecture

## 📋 المتطلبات

- Python 3.8+
- pip

## 🚀 التثبيت والتشغيل

### 1. تثبيت المكتبات المطلوبة



افتح المتصفح على العنوان:

## 🎯 كيفية الاستخدام

### 1. إنشاء حساب
- افتح التطبيق
- اختر "إنشاء حساب"
- أدخل بياناتك
- اختر كلمة مرور قوية

### 2. تشفير ملف
- اسحب الملف إلى منطقة الرفع أو انقر للاختيار
- اختر طريقة التشفير المناسبة
- أدخل كلمة مرور قوية
- (اختياري) أضف مفتاح إضافي للأمان
- انقر "تشفير الملف"

### 3. فك تشفير ملف
- انقر على "فك التشفير" بجانب الملف
- أدخل كلمة المرور والمفتاح الإضافي
- سيتم تنزيل الملف الأصلي تلقائياً

### 4. مشاركة ملف
- انقر على "مشاركة" بجانب الملف
- حدد مدة صلاحية الرابط
- (اختياري) أضف كلمة مرور للمشاركة
- انسخ الرابط وشاركه

## 🔧 الإعدادات المتقدمة

### تغيير حجم الملف الأقصى
في ملف `app.py`:
```python
app.config['MAX_CONTENT_LENGTH'] = 100 * 1024 * 1024  # 100MB
```

### تغيير عدد تكرارات PBKDF2
في ملف `crypto_manager.py`:
```python
self.iterations = 100_000  # زيادة للأمان الأعلى
```

### تغيير منفذ التشغيل
في ملف `app.py`:
```python
app.run(debug=True, host='0.0.0.0', port=5000)
```

## 🛡️ ملاحظات أمنية

1. **كلمات المرور**: استخدم كلمات مرور قوية (12+ حرف، أحرف كبيرة وصغيرة، أرقام، رموز)
2. **المفتاح الإضافي**: يضيف طبقة أمان إضافية - احتفظ به في مكان آمن
3. **النسخ الاحتياطي**: احتفظ بنسخة من قاعدة البيانات `vault.db`
4. **HTTPS**: في الإنتاج، استخدم HTTPS دائماً
5. **الملفات المشفرة**: لا يمكن استرجاع الملفات بدون كلمة المرور الصحيحة
