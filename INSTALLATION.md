# ⚡ تثبيت وتشغيل سريع

## 3 خطوات فقط!

---

## 1️⃣ تحميل المشروع

```bash
git clone https://github.com/MANOUSH71/secure-file-vault.git
cd secure-file-vault
```

**أو** حمل ZIP من GitHub واستخرجه

---

## 2️⃣ تثبيت المكتبات

```bash
pip install -r requirements.txt
```

---

## 3️⃣ تشغيل المشروع

```bash
python app.py
```

---

## 🌐 افتح المتصفح

```
http://localhost:5000
```

---

## 🎉 تم!

الآن:
- سجل حساب جديد
- ارفع وشفر ملفات
- أنشئ فريق
- استمتع!

---

## 🆘 مشاكل؟

### Python مش شغال؟
حمله من: https://www.python.org/downloads/

### البورت 5000 مستخدم؟
غير في `app.py` السطر الأخير:
```python
app.run(debug=True, host='0.0.0.0', port=5001)
```

### المكتبات مش بتتثبت؟
جرب:
```bash
python -m pip install -r requirements.txt
```
