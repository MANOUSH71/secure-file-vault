## 🎮 كيفية الاستخدام

### 1. افتح التوثيق التفاعلي
افتح المتصفح واذهب إلى:
```
http://localhost:8000/api/docs
```

### 2. سجل مستخدم جديد
1. اضغط على **POST /api/v1/auth/register**
2. اضغط **Try it out**
3. أدخل البيانات:
```json
{
  "email": "admin@example.com",
  "username": "admin",
  "password": "Admin123!",
  "first_name": "Admin",
  "last_name": "User",
  "role": "admin"
}
```
4. اضغط **Execute**

### 3. سجل الدخول
1. اضغط على زر **Authorize** في أعلى الصفحة
2. استخدم endpoint **POST /api/v1/auth/token**
3. أدخل:
   - Username: `admin@example.com`
   - Password: `Admin123!`
4. اضغط **Authorize**

### 4. جرب أي endpoint!
الآن يمكنك تجربة جميع الـ endpoints:
- إدارة الأصول (Assets)
- تتبع الثغرات (Vulnerabilities)
- تقييم المخاطر (Risks)
- لوحة التحكم (Dashboard)
- الإشعارات (Notifications)
