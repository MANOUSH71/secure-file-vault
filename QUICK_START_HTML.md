# ⚡ Quick Start - HTML Version

## Get Started in 2 Minutes!

### Step 1: Download Files (30 seconds)
Download these 5 files to a folder on your computer:
- ✅ `index.html`
- ✅ `share.html`
- ✅ `styles.css`
- ✅ `crypto.js`
- ✅ `app.js`

### Step 2: Open Application (10 seconds)
**Double-click `index.html`** - That's it!

The application opens in your browser. No installation needed!

### Step 3: Create Account (30 seconds)
1. Click "Create Account"
2. Fill in your details
3. Click "Create Account"

### Step 4: Encrypt Your First File (1 minute)
1. **Drag & drop** a file into the upload area
2. Choose **AES-256-GCM** (recommended)
3. Enter a **strong password**
4. Click **"🔒 Encrypt File"**

Done! Your file is now encrypted and secure! 🎉

## 🎯 Quick Tips

### Decrypt a File
1. Click "🔓 Decrypt" next to your file
2. Enter your password
3. File downloads automatically

### Share a File
1. Click "🔗 Share" next to your file
2. Set expiration time
3. Copy the link and share it

### Search Files
Type in the search box to filter your files instantly

## 🆘 Quick Troubleshooting

**Can't open the file?**
- Right-click `index.html` → Open With → Choose your browser

**Files not saving?**
- Make sure JavaScript is enabled in your browser

**Forgot your password?**
- Unfortunately, encrypted files cannot be recovered without the password
- This is by design for maximum security

5. Enable GitHub Pages
6. Get your URL: `https://yourusername.github.io/repo-name`

Now you can access it from anywhere!
