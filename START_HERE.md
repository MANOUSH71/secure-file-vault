## 📁 Project Files Overview

### Core Files
- **index.html**: Main application interface
- **share.html**: File sharing page
- **styles.css**: Application styling
- **app.js**: Application logic
- **crypto.js**: Encryption engine
- **analytics.js**: Dashboard statistics
- **audit.js**: Local activity tracking
- **notifications.js**: User alerts
- **requests.js**: File request system
- **teams.js**: Basic team management (local)
