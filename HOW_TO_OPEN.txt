🚀 METHOD 1: DOUBLE-CLICK (EASIEST!)
═══════════════════════════════════════════════════════════════════

   1. Find "index.html" in your folder
   2. Double-click it
   3. It opens in your default browser
   4. Done! ✅


🚀 METHOD 2: RIGHT-CLICK
═══════════════════════════════════════════════════════════════════

   1. Right-click on "index.html"
   2. Select "Open with"
   3. Choose your browser (Chrome, Firefox, Edge, Safari)
   4. Done! ✅


🚀 METHOD 3: DRAG & DROP
═══════════════════════════════════════════════════════════════════

   1. Open your web browser
   2. Drag "index.html" into the browser window
   3. Drop it
   4. Done! ✅


🚀 METHOD 4: BROWSER FILE MENU
═══════════════════════════════════════════════════════════════════

   1. Open your browser
   2. Press Ctrl+O (Windows) or Cmd+O (Mac)
   3. Navigate to your folder
   4. Select "index.html"
   5. Click "Open"
   6. Done! ✅


🚀 METHOD 5: ADDRESS BAR
═══════════════════════════════════════════════════════════════════

   1. Open your browser
   2. In the address bar, type: file:///
   3. Then type the full path to index.html
   
   Example:
   file:///C:/Users/YourName/Desktop/vault/index.html
   
   4. Press Enter
   5. Done! ✅


📱 OPEN ON PHONE/TABLET
═══════════════════════════════════════════════════════════════════

   1. Upload all 5 files to Google Drive or Dropbox
   2. Open Google Drive/Dropbox on your phone
   3. Tap on "index.html"
   4. Select "Open in browser"
   5. Done! ✅

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

   ❌ Problem: File won't open
   ✅ Solution: Make sure all 5 files are in the same folder

   ❌ Problem: Blank page
   ✅ Solution: Check if JavaScript is enabled in browser

   ❌ Problem: Styles not working
   ✅ Solution: Ensure styles.css is in the same folder

   ❌ Problem: Can't encrypt files
   ✅ Solution: Make sure crypto.js and app.js are present

   ❌ Problem: Browser shows error
   ✅ Solution: Try a different browser (Chrome recommended)
