## 🚀 كيفية الاستخدام

### الخطوة 1: تحميل الملفات

حمّل هذه الملفات الـ 5 على جهازك:
- `index.html` - الصفحة الرئيسية
- `share.html` - صفحة المشاركة
- `styles.css` - التنسيقات
- `crypto.js` - مكتبة التشفير
- `app.js` - منطق التطبيق

### الخطوة 2: فتح في المتصفح

**الطريقة 1: النقر المزدوج (الأسهل!)**
1. ابحث عن ملف `index.html`
2. انقر عليه مرتين
3. سيفتح في متصفحك الافتراضي
4. تم! ✅

**الطريقة 2: السحب والإفلات**
1. افتح متصفحك
2. اسحب ملف `index.html` إلى نافذة المتصفح
3. أفلته
4. تم! ✅

### الخطوة 3: إنشاء حساب

1. اضغط "إنشاء حساب"
2. أدخل بياناتك:
   - الاسم الأول
   - الاسم الأخير
   - اسم المستخدم
   - كلمة المرور (8+ أحرف)
3. اضغط "إنشاء حساب"

### الخطوة 4: تشفير الملفات

1. **رفع الملف**:
   - اسحب وأفلت ملف، أو
   - اضغط زر "اختر ملف"

2. **إعداد التشفير**:
   - اختر طريقة التشفير (AES-256-GCM موصى به)
   - أدخل كلمة مرور قوية
   - (اختياري) أضف مفتاح إضافي للأمان

3. **التشفير**:
   - اضغط "🔒 تشفير الملف"
   - انتظر حتى يكتمل التشفير

### الخطوة 5: فك تشفير الملفات

1. ابحث عن ملفك في القائمة
2. اضغط "🔓 فك التشفير"
3. أدخل كلمة المرور والمفتاح الإضافي (إن وجد)
4. اضغط "فك التشفير والتنزيل"
5. سيتم تنزيل الملف تلقائياً

### الخطوة 6: مشاركة الملفات

1. اضغط "🔗 مشاركة" بجانب الملف
2. حدد وقت انتهاء الصلاحية (بالساعات)
3. (اختياري) أضف كلمة مرور للمشاركة
4. اضغط "إنشاء رابط المشاركة"
5. انسخ الرابط وشاركه

## 🔧 التفاصيل التقنية

### طرق التشفير

#### AES-256-GCM
- **الخوارزمية**: معيار التشفير المتقدم
- **حجم المفتاح**: 256 بت
- **الوضع**: Galois/Counter Mode
- **السرعة**: سريع جداً
- **الأمان**: عسكري

#### اشتقاق المفتاح
- **الخوارزمية**: PBKDF2
- **التجزئة**: SHA-256
- **التكرارات**: 100,000
- **الملح**: 16 بايت عشوائي
- **المخرج**: مفتاح 256 بت

### تخزين البيانات

```javascript
هيكل LocalStorage:
├── vault_users         // حسابات المستخدمين
├── vault_files         // الملفات المشفرة
├── vault_current_user  // الجلسة النشطة
└── vault_shares        // روابط المشاركة
```

- تأكد من وجود share.html في نفس المجلد

## 🎓 كيف يعمل

### عملية التشفير

```
1. المستخدم يختار ملف
2. قراءة الملف كـ ArrayBuffer
3. توليد ملح عشوائي (16 بايت)
4. اشتقاق مفتاح من كلمة المرور باستخدام PBKDF2
5. توليد IV عشوائي (12 بايت)
6. تشفير البيانات بـ AES-256-GCM
7. تجميع: [ملح][IV][بيانات مشفرة]
8. ترميز كـ Base64
9. تخزين في LocalStorage
```

### عملية فك التشفير

```
1. استرجاع البيانات المشفرة من LocalStorage
2. فك ترميز Base64
3. استخراج الملح، IV، والنص المشفر
4. اشتقاق مفتاح من كلمة المرور باستخدام PBKDF2
5. فك التشفير بـ AES-256-GCM
6. إرجاع بيانات الملف الأصلية
7. تنزيل إلى جهاز المستخدم
```
